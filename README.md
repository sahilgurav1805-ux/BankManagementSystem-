# Bank Management System

A desktop banking application developed using **Java, Swing, JDBC, and MySQL**.
This project simulates core banking operations with a user-friendly graphical interface.

---

## 🚀 Features

* 🔐 User Login Authentication
* 📝 Account Registration / Signup
* 💰 Deposit Money
* 💸 Withdraw Money
* 📊 Balance Enquiry
* 📄 Mini Statement
* 🔄 PIN Change
* 🗄️ MySQL Database Integration
* 🖥️ Java Swing GUI

---

## 🛠️ Tech Stack

* **Language:** Java
* **GUI:** Swing / AWT
* **Database:** MySQL
* **Connectivity:** JDBC
* **IDE:** IntelliJ IDEA / Eclipse / NetBeans

---

## 📂 Project Structure

* Login.java
* Signup.java
* Deposit.java
* Withdraw.java
* BalanceEnquiry.java
* MiniStatement.java
* PinChange.java
* Conn.java

---

## ▶️ How to Run

1. Clone the repository
2. Open project in your Java IDE
3. Create MySQL database and tables
4. Update database username/password in Conn.java
5. Run Login.java
